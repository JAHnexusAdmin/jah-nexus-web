Direccionamiento IPv4 / IPv6

Gestión completa del espacio de direcciones IP: prefijos, rangos de asignación, IPs individuales y VLANs. Sabe en tiempo real qué IPs están asignadas, cuáles están libres y qué dispositivo las usa.

Asignación de Roles y Permisos

Define quién puede ver, editar o aprovisionar recursos en NetBox. Los ingenieros senior tienen acceso total, el equipo NOC puede consultar pero no modificar. Control de acceso granular por función.

Modelado de Cableado y Conexiones

Documenta cada conexión física: fibra, cobre, patch cords y enlaces WAN. Desde qué puerto de OLT sale el splitter, hasta en qué ONT termina el hilo. Un mapa siempre disponible.

REST API para Automatización

NetBox expone una API REST completa y documentada. Integrala con tus scripts de aprovisionamiento, con Ansible para configuración automatizada o con JAH Sentinel para enriquecer el contexto de los incidentes con datos de inventario físico en tiempo real.

Estrategia de Documentación

De hoja de cálculo a Source of Truth

La migración de tu documentación actual a NetBox es un proceso guiado que realizamos junto a tu equipo técnico.

01
Auditoría de Documentación Existente

Recopilamos y evaluamos toda la documentación actual: hojas de cálculo, diagramas de Visio, archivos de texto y conocimiento tácito del equipo para construir el mapa de migración.

02
Modelado de la Estructura de Datos

Definimos la jerarquía de sitios, racks y dispositivos según la topología real de tu red. Configuramos los tipos de dispositivos, roles y campos personalizados que tu operación requiere.

03
Migración e Importación Masiva

Utilizamos la API de NetBox para migrar la información existente de forma masiva. Los datos se validan automáticamente para eliminar duplicados y errores antes de la importación definitiva.

04
Activación y Capacitación del Equipo

Entrenamos a tu equipo en los flujos de actualización de NetBox. Implementamos las integraciones con JAH Sentinel y Jah-NOC para que la documentación siempre esté al día sin esfuerzo extra.
